# Portfolio personnel

Site statique multi-pages inspire par la direction Figma "Experience Welcome" et personnalise pour le profil de Yacin Bakadir.

Repository cible: https://github.com/bakadiryacin-commits/yacin.bakadir

## A personnaliser

- `index.html`: page d'accueil.
- `portfolio.html`: projets et preuves.
- `use-cases.html`: cas d'usage et methode.
- `case-outbound-nettoyage.html`: prospection outbound automatisee.
- `case-data-enrichment.html`: enrichissement de donnees avec Apps Script.
- `case-seo-automation.html`: outil de redaction et publication SEO.
- `case-effet-home.html`: creation de marque e-commerce.
- `blog.html`: notes marketing.
- `contact.html`: page de contact.
- `cv.html`: CV en ligne.
- `styles.css`: styles communs, responsive et animations.
- `script.js`: compteurs, reveal au scroll et interactions de cartes.

## Ouvrir le site

Ouvre `index.html` dans ton navigateur. Aucun serveur local n'est requis.

## Publier sur GitHub

Depuis ce dossier:

```bash
git init
git branch -M main
git add .
git commit -m "Initial portfolio website"
git remote add origin https://github.com/bakadiryacin-commits/yacin.bakadir.git
git push -u origin main
```

## Deployer sur Vercel

Une fois le projet sur GitHub, importe le repository dans Vercel. Le site est statique: pas de build command, pas de dossier de sortie.
